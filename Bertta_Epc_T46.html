""" ASTEC Quote PDF → Excel + Google Sheets ======================================== Lee un PDF de cotización ASTEC y guarda los datos en: 1. Excel local (igual que antes) 2. Google Sheets (historial acumulado) SETUP (una sola vez) — ver SETUP_GOOGLE_SHEETS.md pip install pdfplumber pandas openpyxl gspread google-auth """ import re import pdfplumber import pandas as pd from datetime import date # ── CONFIGURACIÓN ───────────────────────────────────────────────────────────── CREDENTIALS_PATH = r'C:\Users\backofficemineria\Documents\PHYTON\credentials.json' # Solo el ID del Sheet (parte entre /d/ y /edit en la URL) # URL completa: https://docs.google.com/spreadsheets/d/1bfQL9jxcvGzvDqKhYuYerekwRaGZCp1c1nIQet7NL_0/edit SHEET_ID = '1bfQL9jxcvGzvDqKhYuYerekwRaGZCp1c1nIQet7NL_0' SHEET_TAB = 'Cotizaciones' # ────────────────────────────────────────────────────────────────────────────── HEADER_SHEETS = [ 'Quote#', 'Fecha', 'Quote Date', 'Expires', 'Line', 'Part', 'Description', 'Expected Qty', 'Unit Price', 'Ext. Price', 'Lead Time', 'Notes' ] # Patrón de línea de ítem: ITEM_RE_B = re.compile(r'^(\d+)\s+(\S+)$') QTY_PRICE_RE = re.compile(r'^(\d+\.\d+)\s+([\d,]+\.\d+)\s+([\d,]+\.\d+)') ITEM_RE = re.compile( r'^(\d+)\s+(\S+)\s+(\d+\.\d+)[A-Z]+\s+([\d,]+\.\d+)\s+([\d,]+\.\d+)' ) LEAD_TIME_LINE_RE = re.compile(r'^LEAD\s+TIME\b', re.IGNORECASE) # ── Extracción ───────────────────────────────────────────────────────────────── def extraer_header(texto): """Extrae metadatos del encabezado de la cotización (página 1).""" qn = re.search(r'Quote\s+Number[:\s]+(\S+)', texto, re.IGNORECASE) dt = re.search(r'Date:\s*(\d{1,2}/\d{1,2}/\d{4})', texto) ex = re.search(r'Expires:\s*(\d{1,2}/\d{1,2}/\d{4})', texto) # Lead time global (ej: "LEAD TIME POR CONFIRMAR" o "Lead time: 8 weeks") lt = re.search(r'LEAD\s+TIME[^\n]*', texto, re.IGNORECASE) return { 'quote': qn.group(1) if qn else 'DESCONOCIDO', 'quote_date': dt.group(1) if dt else '', 'expires': ex.group(1) if ex else '', 'lead_time_global': lt.group(0).strip() if lt else '', } def parsear_pdf(ruta_pdf): """ Extrae todos los ítems de la cotización. Devuelve (header_info dict, lista de dicts con los ítems). """ header_info = {} items = [] with pdfplumber.open(ruta_pdf) as pdf: for pg_num, page in enumerate(pdf.pages): texto = page.extract_text() or '' lineas = texto.splitlines() # Encabezado solo en página 1 if pg_num == 0: header_info = extraer_header(texto) print(f" Quote Number : {header_info['quote']}") print(f" Fecha : {header_info['quote_date']}") print(f" Expira : {header_info['expires']}") print(f" Lead Time : {header_info['lead_time_global'] or '(no indicado)'}") # Páginas de totales / T&C no tienen ítems if 'TERMS AND CONDITIONS' in texto: break i = 0 while i < len(lineas): linea = lineas[i].strip() m = ITEM_RE.match(linea) # ── Formato A: todo en una línea "1 PART 3.00EA 26.76 80.28" ── if m: line_num = m.group(1) part = m.group(2) qty = float(m.group(3)) unit_price = float(m.group(4).replace(',', '')) ext_price = float(m.group(5).replace(',', '')) j = i + 1 # ── Formato B: "1 PART" en una línea, qty+precio en otra ── else: mb = ITEM_RE_B.match(linea) if mb: # Buscar la línea de qty+precio en las siguientes líneas qp = None j = i + 1 while j < len(lineas) and qp is None: qp = QTY_PRICE_RE.match(lineas[j].strip()) if qp is None: j += 1 if qp is None: i += 1 continue line_num = mb.group(1) part = mb.group(2) qty = float(qp.group(1)) unit_price = float(qp.group(2).replace(',', '')) ext_price = float(qp.group(3).replace(',', '')) j += 1 # avanzar pasado la línea de qty+precio else: i += 1 continue # ── Capturar descripción (igual para ambos formatos) ── desc_parts = [] lt_item = header_info.get('lead_time_global', '') while j < len(lineas): ln = lineas[j].strip() if ITEM_RE.match(ln) or ITEM_RE_B.match(ln): break if ln == '': j += 1 continue ln_clean = re.sub(r'\s+Discount\s+-[\d,.]+', '', ln).strip() if not ln_clean or re.match(r'^Discount\s', ln_clean): j += 1 continue # Si la línea ES lead time → va a lt_item, NO a description if LEAD_TIME_LINE_RE.match(ln_clean): lt_item = re.sub(r'^LEAD\s+TIME\s*', '', ln_clean, flags=re.IGNORECASE).strip() else: desc_parts.append(ln_clean) j += 1 lt_item = header_info.get('lead_time_global', '') for dp in desc_parts: lm = re.search(r'Lead\s+time[:\s]+(.+)', dp, re.IGNORECASE) if lm: lt_item = lm.group(1).strip() items.append({ 'Line': line_num, 'Part': part, 'Expected Qty': qty, 'Unit Price': unit_price, 'Ext. Price': ext_price, 'Description': ' '.join(desc_parts).strip(), 'Lead Time': lt_item, }) i = j return header_info, items # ── Google Sheets ────────────────────────────────────────────────────────────── def get_sheets_client(): import gspread from google.oauth2.service_account import Credentials scopes = ['https://www.googleapis.com/auth/spreadsheets'] creds = Credentials.from_service_account_file(CREDENTIALS_PATH, scopes=scopes) return gspread.authorize(creds) def ensure_header(worksheet): primera = worksheet.row_values(1) if not primera: worksheet.append_row(HEADER_SHEETS) print(" Header creado en Google Sheets.") def probar_conexion_sheets(): gc = get_sheets_client() sh = gc.open_by_key(SHEET_ID) print("Archivo abierto:", sh.title) print("Pestañas disponibles:", [ws.title for ws in sh.worksheets()]) ws = sh.worksheet(SHEET_TAB) print("Pestaña encontrada:", ws.title) print("Primera fila:", ws.row_values(1)) def subir_a_sheets(header_info, items): try: gc = get_sheets_client() sh = gc.open_by_key(SHEET_ID) ws = sh.worksheet(SHEET_TAB) ensure_header(ws) fecha_hoy = date.today().isoformat() filas = [] for d in items: filas.append([ header_info['quote'], fecha_hoy, header_info.get('quote_date', ''), header_info.get('expires', ''), d['Line'], f"'{d['Part']}", d['Description'], d['Expected Qty'] if d['Expected Qty'] is not None else '', d['Unit Price'] if d['Unit Price'] is not None else '', d['Ext. Price'] if d['Ext. Price'] is not None else '', d['Lead Time'] or '', '', # Notes — rellenar manualmente si se desea ]) ws.append_rows(filas, value_input_option='USER_ENTERED') print(f" ✅ {len(filas)} filas añadidas al Google Sheet (Quote# {header_info['quote']})") except Exception as e: import traceback print(" ⚠️ No se pudo escribir en Google Sheets") print(f" Tipo : {type(e).__name__}") print(f" Repr : {repr(e)}") print(" Traceback:") traceback.print_exc() print(" El Excel local sí se generó correctamente.") print(" Verifica que SHEET_ID sea solo el ID (no la URL completa).") # ── Main ─────────────────────────────────────────────────────────────────────── def procesar_cotizacion(ruta_pdf, ruta_excel): print(f"\n{'─'*55}") print(f" Procesando: {ruta_pdf}") print(f"{'─'*55}") header_info, items = parsear_pdf(ruta_pdf) print(f" Items extraídos: {len(items)}") # Excel df = pd.DataFrame(items, columns=[ 'Line', 'Part', 'Description', 'Expected Qty', 'Unit Price', 'Ext. Price', 'Lead Time' ]) df.to_excel(ruta_excel, index=False) print(f" Excel guardado: {ruta_excel}") # Google Sheets subir_a_sheets(header_info, items) print(f"{'─'*55}\n") if __name__ == '__main__': ruta_pdf = r'C:\Users\backofficemineria\Documents\PHYTON\Cot_Astec_Pdf\Quote_Astec.pdf' ruta_excel = r'C:\Users\backofficemineria\Documents\PHYTON\Cot_Astec_Pdf\cotizacion_procesada.xlsx' procesar_cotizacion(ruta_pdf, ruta_excel) probar_conexion_sheets()
